# 亚马逊广告分析系统：三阶段开发计划

## 当前状态

- ✅ 基础 SKU 表现矩阵 (Inventory vs TACOS)
- ✅ 基础 Mock 数据生成 (3 店铺, 72 SKU)
- ✅ Premium 视觉风格 (Scoped CSS)
- ✅ 基础后端 API (Stores, Matrix)

## 第一阶段：引擎层强化 - 指标与规则建模 (当前进行中)

- [ ] **全指标计算**：在 API 中集成 CTR, CVR, ACOS, ROAS 计算。
- [ ] **利润建模**：结合 COGS 计算广告后净利润率。
- [ ] **健康度评分**：建立全店/店铺维度的健康度评分逻辑。
- [ ] **数据引擎升级**：支持按日期范围聚合数据。

## 第二阶段：应用层扩展 - 多维决策看板

- [ ] **路由重构**：支持 Overview / Matrix / Actions 三大模块。
- [ ] **广告健康总览页**：全店 KPI 仪表盘。
- [ ] **SKU 深度详情页**：钻取查看 30 天趋势图、关键词贡献度。
- [ ] **行动建议中心**：按优先级排列的优化清单。

## 第三阶段：智能层接入 - AI 诊断与决策

- [ ] **Gemini 诊断集成**：将决策逻辑表固化为 AI 提示词。
- [ ] **优先级引擎**：根据亏损和风险自动排序行动项。
- [ ] **策略模拟器**：预测调整竞价对销量和库存的影响。
- [ ] **CSV 数据接入**：支持手动上传广告报告和成本表。
