# Amazon SP-API 参考资料

本文档汇总 Amazon Selling Partner API (SP-API) 的官方参考资料，供开发时查阅。

## 官方文档

- **SP-API 开发者文档 (中文)**
  - [https://developer-docs.amazon.com/sp-api/lang-zh_CN/docs/welcome](https://developer-docs.amazon.com/sp-api/lang-zh_CN/docs/welcome)
  - 包含：API 概览、认证授权、各模块 API 参考、最佳实践等

## API 模型定义

- **SP-API Models (GitHub)**
  - [https://github.com/amzn/selling-partner-api-models](https://github.com/amzn/selling-partner-api-models)
  - 包含：所有 SP-API 的 OpenAPI/Swagger 模型定义
  - 可用于自动生成客户端 SDK

## 常用 API 模块

| 模块 | 用途 |
|------|------|
| **Advertising API** | 广告投放管理、报告 |
| **Catalog Items API** | 商品目录信息 |
| **Orders API** | 订单管理 |
| **Reports API** | 各类业务报告 |
| **Fulfillment API** | FBA 相关操作 |
