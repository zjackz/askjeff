# 亚马逊高阶广告诊断及优化

> 基于《数据+AI双驱动：亚马逊高阶广告诊断及优化20251117》整理

## 1. 广告诊断全流程

### 1.1 诊断核心思路

改善广告前，必须先"确诊"。不要盲目优化，先看病。

**流量漏斗模型**:

1. **搜索量** (没有人搜?) -> 市场需求问题
2. **展现量** (没有人看?) -> 广告投放/排名问题
3. **点击量** (没有人点?) -> 详情页/主图/价格问题
4. **销量** (看了不买?) -> 转化率/评论/详情页问题

**诊断金字塔**:

1. **整体账户**: 宏观健康度
2. **广告商品**: 哪些 ASIN 有问题
3. **广告投放**: 关键词/受众是否精准
4. **客户搜索词**: 实际流量来源

### 1.2 关键诊断指标

| 指标 | 定义 | 诊断意义 |
|------|------|----------|
| **TACOS** | 广告总花费 / 总销售额 | 衡量广告对整体生意的贡献。**TACOS < 毛利率** = 整体盈利。 |
| **ACOS** | 广告花费 / 广告销售额 | 衡量广告本身的投入产出比。 |
| **预计销售月数** | (FBA库存 + 在途) / 月均销量 | **过低(<1)**: 断货风险，需控制广告预算。<br>**过高(>3)**: 滞销风险，需加大广告清库存。 |
| **广告销售贡献比** | 广告销售额 / 总销售额 | 衡量对广告的依赖程度。 |

## 2. 一键诊断实战 (地毯案例)

### 2.1 案例背景

某地毯卖家有 4 个变体 (5x7, 6x9, 8x10, 9x12)，毛利率均为 30%。

### 2.2 诊断数据表

通过整合**广告报告**和**业务报告**，制作如下诊断表：

| SKU | 尺寸 | 广告ACOS | TACOS | 预计销售月数 | 诊断结论 |
|-----|------|----------|-------|--------------|----------|
| **A** | 5x7 | 30.95% | 20.94% | **3.12** | **主力款但库存积压**。TACOS 健康，但库存周转慢。策略：**激进推广**，抢占排名，加速周转。 |
| **B** | 6x9 | 25.92% | **2.62%** | **1.65** | **潜力爆款**。TACOS 极低，自然流量占比高，库存健康。策略：**适当增投**，扩大战果。 |
| **C** | 8x10 | 41.51% | 14.41% | **10.42** | **滞销款**。库存严重积压(10个月)，ACOS 高。策略：**降价清仓 + 广告促销**，首要目标是回笼资金，不看 ACOS。 |
| **D** | 9x12 | 32.50% | 3.36% | **21.48** | **严重滞销**。库存积压最严重(21个月)。策略：**大力清仓**，甚至站外推广。 |

### 2.3 诊断决策逻辑

1. **看库存 (预计销售月数)**:
    * 库存太多 -> 必须打广告清库存，容忍高 ACOS。
    * 库存太少 -> 减少广告，防断货，保利润。
2. **看 TACOS**:
    * TACOS 低于毛利率 -> 整体盈利，有空间增加预算。
    * TACOS 高于毛利率 -> 整体亏损，需严控 ACOS。

## 3. 广告优化策略

### 3.1 优化四部曲

1. **检查 Listing**: 图片、标题、五点、评论。如果 Listing 只有 3 星，先别打广告。
2. **优化投放**:
    * **自动广告**: 跑词，识别高转化词。
    * **手动广告**: 精准打击高绩效词。
3. **优化竞价**:
    * **动态竞价 - 提高和降低**: 适合冲排名、清库存。
    * **动态竞价 - 只降低**: 适合保利润、防超支。
    * **固定竞价**: 适合打精准词、抢 Top of Search。
4. **否定策略**:
    * 高点击低转化 -> 否定。
    * 不相关词 -> 否定。

### 3.2 广告位调整 (Placement)

* **Top of Search (首页首位)**: 点击率高，CPC 高。适合推新品、打排名。
* **Product Pages (商品页面)**: 流量大，转化一般。适合关联流量、竞品拦截。
* **Rest of Search (其余位置)**: 长尾流量。

**调整技巧**:
如果首页首位 ACOS 低且转化好 -> **增加首页首位溢价 (e.g. +20%)**，同时适当降低基础竞价。

## 4. AI 赋能广告

利用 AI 工具 (如 ChatGPT, Claude) 辅助：

1. **拓词**: "我是卖[产品]的，请生成 50 个相关的长尾关键词，按搜索意图分类。"
2. **写文案**: "请为我的[产品]写 5 条吸引人的广告标题 (Headline)，突出[卖点]。"
3. **分析报告**: 将广告报告数据 (脱敏后) 喂给 AI，让它找出表现最差的 10 个关键词。

---

## 5. 常见误区

1. **只看 ACOS**: 忽略了 TACOS 和库存状况。清库存时 ACOS 高是正常的。
2. **频繁调整**: 广告需要时间积累数据 (学习期)，建议至少观察 7-14 天再做大调整。
3. **所有变体混着打**: 不同尺寸/颜色的表现差异巨大 (如案例中的地毯)，应**分组投放**或**重点投放优胜变体**。
